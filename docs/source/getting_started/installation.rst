************
Installation
************

.. code-block:: bash

   # via pip
   pip install mapshader


Setup Mapshader Conda Environment
=================================

.. code-block:: bash

    conda create -n mapshader python=3.8
    conda activate mapshader
    git clone git@github.com:makepath/mapshader.git
    cd mapshader
    pip install -e .
